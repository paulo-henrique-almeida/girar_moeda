 página desenvolvida em JavaScript em que se pode girar uma moeda e contar a quantidade de caras e coroas.

## 🛠 Como Funciona
- Gire a moeda;
- Veja os resultados da quantidade de caras e coroas;
- Reinicie a contagem quando quiser.
